Dating Telegram Bot

Dating Bot — это Telegram бот для знакомств, созданный для помощи пользователям в поиске новых знакомств на основе их анкет и предпочтений. Проект построен на современной архитектуре с использованием aiogram 3.x и базы данных для хранения профилей.
Возможности

    Регистрация и анкета: Создание подробной анкеты (имя, возраст, пол, город, фото, описание, интересы).

    Просмотр анкет: Просмотр анкет других пользователей в формате "лайк/дизлайк" (аналог Tinder).

    Система лайков: Взаимные лайки приводят к созданию "мэтча".

    Уведомления о мэтчах: При возникновении мэтча оба пользователя получают уведомление и ссылку на диалог.

    Личный кабинет: Просмотр и редактирование собственной анкеты.

    Фильтры поиска: Настройка критериев поиска (пол, возраст, город).

    Статистика: Просмотр количества просмотров, лайков и мэтчей.
